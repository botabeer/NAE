لعبه1
صديقك غاب عنك فترة طويلة، ماذا تفعل؟
1. تتواصل معه فورًا
2. تنتظر أن يبادر هو
3. تراقب الموقف بحذر
4. تنساه قليلاً
إذا صديقك احتاج لمساعدة مالية، ماذا تفعل؟
1. تساعده مباشرة
2. تسأله عن التفاصيل أولًا
3. تقدم نصيحة بدلاً من المال
4. ترفض بلطف
صديقك أخطأ بحقك، كيف تتصرف؟
1. تواجهه بصراحة
2. تنتظر أن يعترف هو
3. تحاول التفاهم بهدوء
4. تتجاهل الموقف
إذا اكتشفت أن صديقك يكذب عليك، ماذا تفعل؟
1. تواجهه مباشرة
2. تحاول فهم السبب
3. تتجاهل الموقف
4. تبتعد مؤقتًا
صديقك يريد مشاركة سر مهم، كيف تتصرف؟
1. تستمع جيدًا وتحافظ على السر
2. تعطي نصيحة بعد الاستماع
3. تشارك السر مع شخص موثوق
4. تبتعد عن الموضوع

لعبه2
شعرت بانجذاب لشخص جديد، ماذا تفعل؟
1. تعبر عن مشاعرك فورًا
2. تراقب الموقف قبل الاقتراب
3. تتحدث مع أصدقاء قبل اتخاذ أي خطوة
4. تحتفظ بالمشاعر لنفسك
إذا علمت أن الشخص المعجب بك يرفضك، ماذا تفعل؟
1. تواجه الموقف بصراحة
2. تحاول تقبله بهدوء
3. تبتعد قليلاً
4. تتجاهل الموقف
موعد رومانسي لم يسر كما خططت، ماذا تفعل؟
1. تتصرف بحماس وتحاول تعديل الوضع
2. تظل هادئًا وتحاول الفهم
3. تبتعد لبعض الوقت
4. تترك الأمور للمصادفة
إذا شعرت بالغيرة، كيف تتصرف؟
1. تناقش الموضوع مباشرة
2. تهدأ أولًا قبل الكلام
3. تراقب الشخص الآخر
4. تتجاهل شعورك
الشخص المعجب بك يرسل إشارات غير واضحة، ماذا تفعل؟
1. تسأله مباشرة
2. تنتظر ليتضح الموقف
3. تحاول تفسير الإشارات بنفسك
4. تتجاهل الإشارات

لعبه3
إذا واجهت فرصة عمل كبيرة، كيف تتصرف؟
1. تخطط للقبول فورًا
2. تدرس الإيجابيات والسلبيات
3. تستشير شخص موثوق
4. تنتظر الأفضل
حلمت بمستقبل غير واضح، ماذا تفعل؟
1. تضع خطة واضحة
2. تبحث عن نصائح وتجارب الآخرين
3. تراقب الوضع
4. تترك الأمور تتطور طبيعيًا
ترغب بتحقيق هدف كبير، كيف تبدأ؟
1. تتخذ خطوات جريئة
2. تخطط بدقة
3. تبحث عن دعم من الآخرين
4. تبدأ ببطء دون ضغط
تواجه قرار مصيري للمستقبل، ماذا تفعل؟
1. تتخذ القرار بسرعة
2. تحلل الخيارات بعناية
3. تطلب مشورة
4. تؤجل القرار
لديك فرصة للسفر لمكان جديد، ماذا تفعل؟
1. تحجز فورًا
2. تفكر في التوقيت والفرص
3. تستشير أشخاص سبقوا التجربة
4. تتركه مؤقتًا

لعبه4
قابلت حب قديم بعد سنوات، ماذا تفعل؟
1. تتحدث معه بحرارة وتسترجع الذكريات
2. تحاول معرفة أخبار حياته بهدوء
3. تتصرف بشكل طبيعي وكأن لا شيء حدث
4. تبتعد لتجنب المشاعر
ذكرياتك مع حب قديم تعود فجأة، كيف تتصرف؟
1. تسترجع التفاصيل وتستمتع بالذكرى
2. تحاول تجاهلها والتركيز على الحاضر
3. تشاركها مع صديق مقرب
4. تكتب عنها في دفتر أو رسالة لنفسك
إذا علمت أن حبك القديم مرتبط الآن بشخص آخر، ماذا تفعل؟
1. تتقبل الأمر بصعوبة ولكن بتحضر
2. تشعر بالغيرة وتحاول التواصل
3. تبتعد وتفكر في مستقبلك
4. تحاول أن تكون صديقًا له/لها
حب قديم أرسل لك رسالة غير متوقعة، كيف تتصرف؟
1. ترد فورًا بحماس
2. تنتظر وتحلل محتوى الرسالة
3. تكتب رد قصير ومحايد
4. تتجاهل الرسالة
وجدت فرصة للقاء حب قديم، ماذا تفعل؟
1. توافق فورًا وتستعد للقاء
2. تفكر في التوقيت والمكان المناسب
3. تتواصل أولًا لمعرفة النوايا
4. ترفض اللقاء للحفاظ على مشاعرك

لعبه5
خلاف مع أحد الأهل، كيف تتصرف؟
1. تواجه المشكلة مباشرة
2. تناقش بهدوء
3. تنتظر حتى يهدأ الجو
4. تتجنب الموضوع
طلب منك أحد الأهل نصيحة، ماذا تفعل؟
1. تقدم النصيحة مباشرة
2. تناقش الخيار معه أولًا
3. تعطي رأيًا عام
4. تترك القرار له
شعرت بالضغط من توقعات الأهل، ماذا تفعل؟
1. تتحدث عن مشاعرك
2. تحاول التكيف
3. تتجاهل الضغط
4. تنسحب مؤقتًا
أحد أفراد العائلة في أزمة، ماذا تفعل؟
1. تساعد فورًا
2. تدرس الوضع قبل التدخل
3. تقدم نصيحة فقط
4. تبتعد قليلاً
ترغب في تحقيق شيء ضد رغبة العائلة، ماذا تفعل؟
1. تتخذ القرار بثقة
2. توازن بين رغبتك ورغبتهم
3. تستشير شخص مقرب
4. تنتظر الوقت المناسب

لعبه10
تواجه موقف صعب، كيف تتصرف؟
1. تواجهه فورًا
2. تخطط للتعامل معه
3. تطلب نصيحة
4. تبتعد
خيار مهم يتطلب قرار سريع، ماذا تفعل؟
1. تتخذ القرار مباشرة
2. تزن الخيارات بعناية
3. تطلب رأي شخص موثوق
4. تؤجل
شخص يضغط عليك، كيف تتصرف؟
1. تواجهه بصراحة
2. تهدأ وتفكر
3. تتجنب المواجهة
4. تبتعد مؤقتًا
فرصة غير متوقعة، ماذا تفعل؟
1. تنتهزها فورًا
2. تدرس المخاطر
3. تسأل شخص ذو خبرة
4. تتركها
موقف غير متوقع، كيف تتصرف؟
1. تحل المشكلة مباشرة
2. تخطط خطوة خطوة
3. تبحث عن دعم
4. تراقب الوضع
